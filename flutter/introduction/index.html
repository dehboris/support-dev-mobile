Introduction

Qu'est ce que Flutter ?

Flutter est framework ou SDK (Software Development Kit) opensource basé sur le langage Dart permettant le développement des applications mobile à la fois sur Android et iOS créé par Google.

Ce framework est utilisé pour tout ce qui est interface utilisateur.

En Flutter, l'interface se construit à l'aide de widgets. Chaque widget permet de décrire ce à quoi leur vue doit ressembler en fonction de leur configuration et de leur état.

(Almost) Everything is a Widget

Qu'est ce que Dart ?

Dart (initialement appelé Dash) est un langage de programmation orienté objet developpé par Google.

Dart peut être utilisé pour de la programmaion côté serveur, ainsi que le développement d'applications mobiles (via l'API Flutter)

Il est facile à comprendre et à prendre en main et se rapproche des langagues tels que Java ou C# par exemple.
