Les variables

La structure de base d'une variable est la suivante

[modifier] type name [= value];

  • type est le type de la variable.
  • name est le nom de la variable.
  • value est la valeur de la variable.

[=value] signifie que cette partie est optionnelle, dans ce cas la variable est automatiquement initialisé à null

Les Types primitifs

Les types primitifs ou explicites sont utilisés pour désigner le type que la variable doit prendre lors de sa déclaration.

Les types supportés par le langage Dart sont :

Type Description
int Les Entiers
double Les nombres à virgules
num Entier ou nombre à virgule
bool Booléen
String Chaîne de caractères
List Tableau
Map Tableau associatifs

Exemple

    String voiture = 'Lexus'; // Déclaration d'une variable de type chaîne de caratères
    int taille = 10;                
    double poids = 20.5;      // Déclaration d'une variable de type entier
    bool estRouge = false;    // Déclaration d'une variable de type booléen

Inférence de Types (Type inference)

L'inférence de types est un mécanisme qui permet à un compilateur ou un interpréteur de rechercher automatiquement les types associés à des expressions, sans qu'ils soient indiqués explicitement dans le code source.

Le mot clé var

L'utilisation du mot clé var permet de déclaré des variables sans précisé le type qui sera connu du programme lors de son initialisation. La modification du contenu de ces variables sont possible en donnant le même type que lors de l'initialisation.

Après l'initialisation, la variable ne peux plus changer de type avant la fin de l'exécution du programme.

Exemple

    var voiture = 'Lexus'; // Déclaration d'une variable de type chaîne de caratères
    var taille = 10;                
    var poids = 20.5;      // Déclaration d'une variable de type entier
    var estRouge = false;    // Déclaration d'une variable de type booléen

Le mot clé dynamic

L'utilisation du mot clé dynamic permet de déclaré des variables de n'importe quel type.

Les modificateurs de variable

  • Lorsque le mot clé final est utilisé lors de la déclaration des variables, elles ne peuvent plus être modifiés après leurs initialisation.

  • Lorsque le mot clé const est utilisé lors de la déclaration des variables, ces variables doivent être initialisé et ne peuvent plus être modifiés.

String ou chaînes de caractères

L'interpolation de chaîne (String Interpolation)

L'interpolation de chaînes permet de combiner des variables et du texte pour former une chaîne. On l'utilise le symbole $ devant le nom de la variable dans la chaîne de caractères.

void main(){
    String name = 'Freddy';

    print("Je me nomme $name et mon prénom compte ${name.length} lettres.");
}

Le résultat sera

Je me nomme Freddy et mon prénom compte 6 lettres.

Concaténation

La concaténation est la somme de deux variables

void main() {
  String name = 'Mathis';
  print(name + 'à fin.');
}

Résultat

Mathias à fin.
